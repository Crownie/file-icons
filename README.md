## Installation

Npm

```shell script
npm i @crownie/file-icons
```

Or<br>

Yarn

```shell script
yarn add @crownie/file-icons
```

## Usage

```typescript jsx
import FileIcon from '@crownie/file-icons';

// use mimetype
<FileIcon type='image/jpeg'/>

// or file extension
<FileIcon type='jpeg'/>

<FileIcon type='mp3'/>

```
